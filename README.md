## 运行项目

``` bash
# 安装依赖
npm install

# 在localhost:8080开始调试
npm run dev

# 建立成品版本
npm run build

```

## 部署
``` bash
#请使用
npm build

#生成成品后，在dist文件夹内
#请将static文件夹部署在域名代理目录的根目录下

```
